QUICK FIX FOR FIRESTORE PERMISSIONS ERROR

Copy and paste this into Firebase Console → Firestore Database → Rules:

rules_version = '2';
service cloud.firestore {
  match /databases/{database}/documents {
    match /{document=**} {
      allow read, write: if request.auth != null;
    }
  }
}

Then click "Publish"

This allows any authenticated user to read/write data.
Your app will handle ensuring users only access their own data.
